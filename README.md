## 🚀 What this app does

obsidian_vault_pipeline helps you process an Obsidian vault with LLM tools. It can take content from sources like Pinboard and Obsidian Clipper, then sort and prepare it for use in your vault.

Use it when you want a simple way to collect notes, clean them up, and keep them in one place.
